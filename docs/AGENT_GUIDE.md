# Agent Guide (sundae)

## Session start

1. Read `AGENTS.md` (especially “Project Links” + workflow defaults).
2. Read `docs/PROGRESS.md`, `docs/HANDOFF.md`, `docs/DECISIONS.md`.
3. Run the fastest relevant checks (usually `pnpm lint` + `pnpm typecheck`).
4. Pick one task; keep scope tight.

## Session end

1. Update `docs/HANDOFF.md` with current state.
2. Update `docs/PROGRESS.md`.
3. Leave clear next steps.
